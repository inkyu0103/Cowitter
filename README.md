## 3. git Flow

- **feat** : 새로운 기능 추가
- **fix** : 오류 해결
- **refactor** : 내용 변경은 없이, 코드가 바뀌는 경우
- **format** : 간단한 코드 변경
- **style** : 스타일 바뀌는 경우
- **doc** : readmd 문서 정리
- **chore** : package.json , tsconfig.json 등등...
